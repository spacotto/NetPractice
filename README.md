## Instructions

Download the project resources archive: 

```bash
net_practice.1.9.tgz
```

Extract the archive content:

```bash
tar -xvzf net_practice.1.9.tgz
```

Access the local web server to deliver NetPractice’s web pages

```bash
./run.sh
```

Select the Training or Evaluation and enter your intranet login.
